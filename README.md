🚀 Expense Tracker (Django)

A full-stack Expense Tracker web application built from scratch using Django to manage and visualize daily spending.

✨ Features
Add, edit, and track expenses

Daily, weekly, and monthly summaries

Budget-based progress indicators

Interactive charts (category & daily trends)

Modern dark UI with responsive design

🛠️ Tech Stack

Backend: Django, Python

Frontend: HTML, Tailwind CSS, JavaScript

Charts: Chart.js

📊 Highlights

Clean and intuitive dashboard

Real-time data visualization

Consistent and user-friendly UI
